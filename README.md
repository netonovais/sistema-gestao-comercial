# sistema-gestao-comercial

Sistema desenvolvido como projeto acadêmico para a disciplina de [...].

## Tecnologias
- Java 21
- Spring Boot 3
- Spring Data JPA
- MySQL
- JWT

## Arquitetura
O projeto segue a Arquitetura em Camadas:
- **Apresentação** – Interface web ou Swing
- **Controller** – API REST (`br.com.sgc.controller`)
- **Service** – Regras de negócio (`br.com.sgc.service`)
- **Domain** – Entidades e repositórios (`br.com.sgc.domain`)
